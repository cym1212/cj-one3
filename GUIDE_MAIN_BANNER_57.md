## 🚀 사용 방법

### 1. props.js 파일 임포트하기

#### CommonJS 방식
```javascript
const { editorProps, componentInfo, mergeWithEditorProps } = require('./props.js');
```

#### ES6 모듈 방식
```javascript
import { editorProps, componentInfo, mergeWithEditorProps } from './props.js';
// 또는
import editorProps from './props.js';
```

### 2. React 컴포넌트에서 사용하기

```javascript
import React from 'react';
import { mergeWithEditorProps } from './props.js';

const MyCustomSkin = (props) => {
  // 에디터 props와 전달받은 props를 병합
  const finalProps = mergeWithEditorProps(props);
  
  return (
    <div className="my-custom-skin">
      {/* finalProps를 사용하여 컴포넌트 렌더링 */}
      <h2>{finalProps.title || '제목'}</h2>
      <p>{finalProps.description || '설명'}</p>
      {/* ... */}
    </div>
  );
};

export default MyCustomSkin;
```

### 3. Props 확장하기

```javascript
import { editorProps } from './props.js';

// 에디터 props를 기반으로 새로운 props 정의
const extendedProps = {
  ...editorProps,
  // 추가 속성
  customColor: '#ff0000',
  enableAnimation: true,
  // 에디터 props 오버라이드
  title: editorProps.title || '기본 제목'
};
```

## 📝 Props 타입 정의 (TypeScript)

TypeScript를 사용하는 경우, 다음과 같이 타입을 정의할 수 있습니다:

```typescript
// props.d.ts
export interface EditorProps {
  skin?: string;
  // 여기에 다른 props 타입을 추가하세요
  [key: string]: any;
}

export interface ComponentInfo {
  id: string;
  type: string;
  skinId: string;
  skinName: string;
  version: string;
}

export declare const editorProps: EditorProps;
export declare const componentInfo: ComponentInfo;
export declare function mergeWithEditorProps(customProps?: Partial<EditorProps>): EditorProps;
```

## 🛠️ 개발 팁

1. **Props 우선순위**: `mergeWithEditorProps` 함수에서 props 병합 순서를 조정하여 우선순위를 설정할 수 있습니다.

2. **기본값 설정**: 에디터에서 설정되지 않은 props에 대해 기본값을 제공하세요.
   ```javascript
   const value = finalProps.someValue || 'defaultValue';
   ```

3. **Props 검증**: PropTypes나 TypeScript를 사용하여 props를 검증하세요.
   ```javascript
   import PropTypes from 'prop-types';
   
   MyCustomSkin.propTypes = {
     title: PropTypes.string,
     description: PropTypes.string,
     // ...
   };
   ```

4. **조건부 렌더링**: props 값에 따라 다른 UI를 렌더링하세요.
   ```javascript
   {finalProps.showHeader && <Header {...finalProps} />}
   ```
